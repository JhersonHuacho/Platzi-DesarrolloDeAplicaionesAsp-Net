-- atajos 

* Para copiar y pegar la misma linea debajo podemos usar el atajo:
    Shift + Alt + ⬇

* Se puede usar el atajo:
    alt + ↑
    alt + ↓
  para mover la linea seleccionada.

* seleccionar una raya
  alt + shift + presionando click
